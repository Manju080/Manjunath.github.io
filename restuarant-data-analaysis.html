Platform Overview

An enterprise-grade, end-to-end data engineering and machine learning pipeline built to process the massive 4.5+ GB Yelp Open Dataset. This platform transforms millions of raw, unstructured reviews and business logs into actionable, predictive business intelligence visualized through a dynamic Streamlit web application.

Core Engineering Achievements:

  • Distributed ETL Pipeline: Designed a Medallion architecture (Bronze/Silver/Gold) using Apache PySpark to ingest, clean, and feature-engineer millions of JSON records into compressed Parquet data clusters.
  • Predictive ML Modeling: Engineered XGBoost classifiers to predict customer churn risk, alongside Prophet/ARIMA models for financial time-series forecasting of restaurant growth.
  • Workflow Orchestration & Tracking: Automated the entire ETL and ML training lifecycle using Prefect workflows, while meticulously logging model artifacts, hyperparameter tuning, and performance metrics via MLflow.
  • Automated CI/CD Deployment: Fully containerized the Streamlit application and PySpark dependencies using Docker. Orchestrated a continuous integration pipeline via GitHub Actions that performs automated testing, security scanning, and seamless live deployment via Render Webhooks.

Orchestrated Architecture Flow

ETL Orchestration (Prefect)

  • Bronze Layer (Raw): Prefect triggers automated workflows to ingest massive, unstructured Yelp JSON payloads (Businesses, Reviews, Users).
  • Silver Layer (Cleaned): Distributed PySpark environments orchestrate data standardization, handling timestamp parsing policies and rigorous schema validation in parallel.
  • Gold Layer (Features): Advanced feature engineering pipelines aggregate billions of data points into specialized, ML-ready Parquet datasets using complex Window functions.

Analytical Domains (MLflow)

  • CRM Intelligence: Orchestrated XGBoost models pipeline the Gold Parquet files to actively predict and log customer churn-risk probabilities.
  • Financial Forecasting: Supervised pipelines fuse text-mining NLP (TextBlob) with Prophet/ARIMA models, extracting sentiment polarities to project 12-month revenue growth.
  • Artifact Registry: Performance metrics, hyperparameters, and finalized predictive models are continuously versioned into the centralized MLflow tracking server.

Deployment Config (Docker/CI)

  • Container Ecosystem: The architecture dynamically links the Streamlit frontend with PostgreSQL MLflow backends seamlessly using localized Docker Compose networks.
  • Automated CI/CD: GitHub Actions YAML workflows orchestrate PyTest validation and Bandit security scans against all new repository pushes to the main branch.
  • Cloud Webhooks: Upon passing isolated tests, the pipeline automatically pings Render's deployment REST APIs to execute zero-downtime production builds of the interactive BI interface.

DevOps & Deployment Strategy:

  • Memory-Optimized Containerization: Engineered a localized Docker Compose footprint that artfully restricts Java Py4J gateway memory spikes, ensuring the data pipeline successfully compiles locally without triggering memory faults.
  • Intelligent Cloud Fallbacks: Engineered a dynamic data-loading pipeline that detects the hosting environment. The app processes the massive local 6.5 GB dataset natively, but gracefully falls back to a bundled 1,000-row parquet sample when compiled in the cloud to bypass GitHub storage constraints.
  • Continuous Integration: Authored rigorous GitHub Actions YAML pipelines executing PyTest, Flake8 linting, and Bandit vulnerability scans before signaling Render's REST API hooks for zero-downtime production builds.

Business Impact & Tangible Outcomes

Proactive Churn Mitigation

Transitioned customer retention strategies from reactive to predictive. The XGBoost model calculates exact churn probabilities per user, allowing businesses to execute targeted re-engagement campaigns for high-risk, high-value "Champion" users.

Data-Driven Forecasting

Eliminated guesswork in operational planning. By leveraging Prophet's seasonal additive modeling, restaurants gain data-backed 12-month projections of organic review volume, growth rates, and popularity trajectories.

Automated Sentiment Discovery

Transformed millions of unstructured textual complaints and praises into quantifiable metrics. The TextBlob integration instantly identifies operational friction points across thousands of reviews without manual reading.

Ultimately, this zero-touch, containerized platform serves as a blueprint for how massive, siloed data repositories can be automatically refined into accessible, highly persuasive business intelligence utilizing state-of-the-art MLOps principles.
